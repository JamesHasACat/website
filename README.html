What is iCreate Pro?

iCreate Pro is a jailbreak-free Geometry Dash mod menu for iOS. iCreate Pro contains many mods, some of the highlights being:

  • Noclip
  • Speedhack (with audio)
  • External songs
  • No Object Limit
  • Copy Hack
  • Practice song hack
